Synchronisez votre progression sur tous vos appareils via GitHub Gist. Si non configuré, le jeu fonctionne normalement en local.

❓ Comment configurer GitHub Gist ?
Étape 1 — Créer un Gist
Rendez-vous sur gist.github.com (compte GitHub gratuit requis).
Filename : save.json • Contenu : {} • Cliquez "Create secret gist".
Copiez l'ID dans l'URL : gist.github.com/votrenom/cet-id-ici

Étape 2 — Créer un token GitHub
1. Sur GitHub, cliquez sur votre avatar en haut a droite → "Your GitHub profile"
2. Sur votre page profil, cliquez a nouveau sur votre avatar en haut a droite
3. Cette fois, cliquez sur "Settings" dans la liste
4. Dans le menu de gauche, descendez tout en bas et cliquez sur "Developer settings"
5. Cliquez sur "Personal access tokens" puis "Tokens (classic)"
6. Cliquez sur "Generate new token" → "Generate new token (classic)"
7. Remplissez les champs :
  • Note : LOL Esport Manager
  • Expiration : No expiration
  • Permission : cochez uniquement la case gist
8. Cliquez "Generate token" en bas de page
9. Copiez immédiatement le token affiche (commencé par ghp_) — il ne sera plus visible après avoir quitte la page.

Étape 3 — Configurer dans le jeu
Collez l'ID du Gist et le token dans les champs ci-dessus, puis cliquez "Enregistrer la configuration".
